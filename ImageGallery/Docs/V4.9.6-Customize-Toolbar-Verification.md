## 用户可见效果

1. **右键 toolbar → "Customize Toolbar..."** 菜单可用
2. Customize 面板：显示 8 个 items（带 label），可拖入/拖出 toolbar
3. 自定义状态自动保存到 UserDefaults（autosavesConfiguration）
4. 重启 app 后保留用户的 toolbar 配置

## macOS 系统行为

- `allowsUserCustomization = true` → 系统在 View 菜单加 "Customize Toolbar..." 项
- `autosavesConfiguration = true` → 用户拖动 items 后自动保存
- `displayMode = .iconOnly` → Customize 面板只显示 icon（不显示 label）——这与 Photos.app 一致
